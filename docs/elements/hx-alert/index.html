The custom <hx-alert> element provides a user with an "at a glance" status of a given content section.

Permitted Parents any Permitted Children in this order:
  1. an <hx-alertdescription> element
  2. an optional <hx-alertaction> element
{% endblock %} {% block attributes %}
aria-live (String [polite])
Automatically set by <hx-alert> when attached to the DOM.
closable
Determines if the alert can be closed.
open
Determines if the alert is open.
role (String [status])
Automatically set by <hx-alert> when attached to the DOM.
{% endblock %} {% block properties %}
open (Boolean [true])
Opens the alert.
closable (Boolean [false])
Sets the alert to be closable.
{% endblock %}
